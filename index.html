¿Qué es Javascript y en qué se asemeja a Java?

JavaScript es un lenguaje de programación de alto nivel, interpretado, con tipado dinámico y débilmente tipado. Se utiliza principalmente para agregar interactividad a páginas web. Aunque su nombre lleva 'Java', JavaScript y Java son dos lenguajes distintos y no están relacionados directamente, aunque comparten ciertas similitudes en la sintaxis y en algunos conceptos
